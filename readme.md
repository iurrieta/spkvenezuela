# spkvenezuela
#### Version 1.0

## Requirements

>>>
*	PHP  5.6.4 >=
*	OpenSSL PHP Extension
*	PDO PHP Extension
*	Mbstring PHP Extension
*	Tokenizer PHP Extension
*	XML PHP Extension
*	Composer
*	Laravel   5.4
>>>

## Instalation

>>> git clone https://github.com/iurrieta/spkvenezuela.git

>>> composer update 

>>> php artisan key:generate 

>>> php artisan migrate
